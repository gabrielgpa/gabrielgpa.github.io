iNeed

APPLICATION TO CALCULATE THE AVERAGE ENOUGH TO BE APPROVED

iNeed is an application that helps students measure which grade they need to pass a course. It is useful to keep the student attentive to the notes, to avoid their disapproval. Through a mobile application, the student can get the information quickly and agile, wherever he is.
iNeed was implemented using PHP technologies for the backend (API) and native Android for the creation of the mobile application.
